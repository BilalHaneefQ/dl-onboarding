Under The Hood

How OpenClaw Works

OpenClaw is an open-source, self-hosted AI agent framework — the operating system that runs intelligent agents on your own machine.

  • Each agent has a SKILL.md — plain Markdown instructions defining behavior
  • Multiple specialist agents run in parallel, each owning their domain
  • All data stays on your machine — nothing reaches a third party
  • 3,700+ community skills available on ClawHub

Current Limitation

Single User Today.
Multi-User Tomorrow.

NOW — STANDARD OPENCLAW

One workspace, one user. All agents share the same memory and credentials. Multi-user causes data bleed.

NEXT — OPENCLAW MULTITENANT

Per-user isolated containers. Each user has their own Google token, memory, and session. Deploy once, onboard unlimited users.

# Standard OpenClaw
user_A → workspace/
user_B → workspace/ # same! ❌

# Multitenant
user_A → containers/A/ ✓
user_B → containers/B/ ✓
user_C → containers/C/ ✓

Agent logic, skills, and workflows stay exactly the same. Only the infrastructure layer changes.

Technical Decision

Why We Moved from
WhatsApp to Discord

Approach Cost Legal Reliability Status
WhatsApp Business API $0.01–0.08/msg ✓ Legal Stable Post-POC
Baileys Protocol Free ⚠ Illegal Unreliable Dropped
Discord Bot API Free ✓ Legal Stable ✓ Current

We started with Baileys (OpenClaw's built-in WhatsApp support). WhatsApp detected the automation, blocked connections, and dropped replies. We switched to the official Discord Bot API — free, stable persistent WebSocket, zero ban risk. For production with real users, the WhatsApp Business API is the right path.

Key Takeaways

What I Learned

Prompt engineering > code

SKILL.md files in plain Markdown define agent behavior more reliably than traditional code for AI workflows.

Spec before implementation

SDD caught the gog attendee-wipe bug before it shipped. Spec first saves debugging time later.

Verify before speccing

gog --remote state is in-memory only — discovered during validation, not design. Always run the command first.

Official APIs > unofficial

Discord Bot API is free, stable, legal. Baileys WhatsApp isn't. Choose reliability over convenience.
