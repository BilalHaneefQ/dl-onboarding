# Project Overview — Omnio

## Project Name
Omnio

## Problem
Professionals waste significant time context-switching between Gmail, Google Calendar, and social media platforms to handle routine tasks — reading emails, scheduling meetings, and posting updates. This fragmentation kills focus and productivity, especially when these tasks demand quick action throughout the day.

## Solution
A multi-agent AI system controlled entirely through a single Discord DM. Built on OpenClaw, the system uses an AI orchestrator that interprets natural language commands and delegates tasks to specialized AI agents — each purpose-built for a specific domain:

- **Email Agent** — reads incoming Gmail, drafts context-aware replies using AI, and waits for user confirmation before sending
- **Calendar Agent** — reschedules existing Google meetings and creates new ones with specified attendees, all via a simple message
- **Meet Agent** — creates Google Meet links, attaches them to calendar invites, notifies attendees

The AI layer understands intent, handles ambiguity, and confirms actions before executing — making it feel like a smart assistant rather than a rigid bot.

## Goal
Deliver a working proof of concept within 2 weeks that demonstrates how AI agents, orchestrated through a Discord DM interface, can replace manual multi-app workflows. The project will validate whether natural language alone is sufficient to manage professional communication, scheduling, and Google Meet — serving as a foundation for broader AI automation initiatives within the company.

## Phase
POC (Proof of Concept) — internal testing exercise for Disrupt Labs.
