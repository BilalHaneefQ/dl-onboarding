# Cost Breakdown

## One-Time Setup Costs
- OpenClaw: **Free** (open source)
- Google APIs (Gmail, Calendar, Meet): **Free**
- Google Cloud Console setup: **Free**

## Recurring Monthly Costs

| Item | Cost | Notes |
|---|---|---|
| LLM API (Claude Opus 4.7) | ~$10–30/month | Depends on usage volume |
| LLM API (DeepSeek V3 — budget option) | ~$2–5/month | Cheaper, slightly less capable |
| VPS Hosting | ~$5–10/month | If not running on own machine |
| X/Twitter API (future) | $0.15 per post | Pay-per-use since Feb 2026 |
| WhatsApp Business API (post-scale) | ~$0.01–0.08 per conversation | Only needed after ~50 users |

## POC Estimated Monthly Cost

| Scenario | Cost |
|---|---|
| Run on own machine + budget LLM | ~$5–10/month |
| VPS + mid-range LLM | ~$15–25/month |
| OpenClaw Cloud (fully managed) | ~$59/month |

## What is Truly Free
- OpenClaw software
- Google Calendar API
- Gmail API
- Google Meet API
- Google Cloud Console (APIs + OAuth setup)
